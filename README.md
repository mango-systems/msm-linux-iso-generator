# Mango Linux ISO Generator and Customiser

## Instructions
- Clone the repo
- customise config (create-iso-from-scratch.sh) and its variables

## Dependencies:
    - binutils
    - debootstrap
    - squashfs-tools
    - xorriso
    - grub-pc-bin
    - grub-efi-amd64-bin 
    - mtools
